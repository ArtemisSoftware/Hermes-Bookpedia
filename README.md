# Hermes-Bookpedia
Book application, where we can browse books

## The Brief

App that searches books from the api openlibrary


## Architecture & Libraries
    - MVI
    - Ktor
    - ROOM Database
    - Dependency Injection - Koin
    - Kotlin Coroutines
